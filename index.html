Introducing

Scent

Scent is a final year project done by SIM-UOW students in group SS16/3F.
It is a network sniffer app for Android paired with an analyser web app.

Sniff packets passing through the network on mobile app.

Upload captured packets of each session on cloud storage.

Analyse each packet and view statistics on web app.

Filter each sessions to work on a subset of the data.

Decode raw data of each packet to human readable text.
